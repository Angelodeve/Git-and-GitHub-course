12_GIT PULL REBASE

Esto se utilizara cuando en github y en git hacemos cambios en la misma linea, y tendremos que usar en git pull

  1. git config --global pull.rebase true Esto se hace una sola vez ya que estamos haciendo una configuracion global
  2. git pull
  3. LUEGO TENEMOS QUE AHCER UN COMMIT (git add . , git commit -m "") ponemos el comando git status
    y aparecera INTERACTIVE REBASE IN PROGRESS Entonces como terminamos el rebase pues con el siguiente comando
  4. git rebase --continue con esto culminamos el rebase
